# goodpicker

## Install

### 1. Backend

- python3
- pipenv  
  `sudo pip3 install pipenv`
- virtualenv  
  `sudo pip3 install virtualenv`
- pew  
  `sudo pip3 install pew`
```
pipenv shell
pipenv install
```
### 2. Frontend
- nodejs 
 ```
 sudo apt update
 sudo apt install nodejs
 ```
- npm 
 `sudo apt install npm`
- yarn 
 `sudo npm i -g yarn`
```
cd frontend
yarn
```

## Start

### 1. Backend
**At project root**
```
pipenv shell
cd backend
python manage.py runserver
```

### 2. Frontend

```
cd frontend
npm start
```
